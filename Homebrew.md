# Package Managers
## Homebrew
List installed packages
```sh
brew list
```
Information about a package
```sh
brew info certifi
```
Uninstall a package
```sh
brew uninstall semgrep
```
Install a package
```sh
brew install certifi
```
Update homebrew itself
```sh
brew update
```
Upgrade all installed packages
```sh
brew upgrade
```
Upgrade a package
```sh
brew upgrade certifi
```
